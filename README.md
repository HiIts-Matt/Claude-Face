# Claude Face

A VS Code extension that shows an animated pixel-art crab that reacts to what Claude Code is doing.

## Setup

1. Install the extension
2. Open the command palette and run **Claude Face: Set up hooks** to wire it into Claude Code
3. The face appears automatically and updates as Claude works

## States

- **Idle** — bobbing crab
- **Thinking** — upward gaze with thought bubble building above
- **Reading** — downward gaze at a large file
- **Writing** — pencil sweeping across the page
- **Running command** — crab faces a terminal, green text types across
- **Success** — big smile, sparkles, green tick
- **Error** — crossed eyes, frown, tongue (minesweeper tribute)
- **Uncertain** — darting eyes, wavy mouth
- **Actually** — raised claw, exclamation mark

## Requirements

[Claude Code](https://claude.ai/code) must be installed and running.
